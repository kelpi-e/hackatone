Название: Минимальная сумма под массива
Описание: Напишите функцию min_subarray_sum(nums, k), которая возвращает минимальную сумму подмассива длины k в списке nums.
Ограничения:
1 ≤ длина списка ≤ 10^5
1 ≤ k ≤ длина списка
-10^4 ≤ элемент списка ≤ 10^4
Пример:
Вход: nums=[3, -4, 2, 1, -6, 5], k=3
Выход: -3
